  • PaintJS (Desktop Only)

    March 2021

    PaintJS is a painting app made using VanillaJS and Canvas2D API

    Features

    • Draw on Canvas using mouse
    • Control pen size easily with range type input
    • Has two modes - Fill & Paint. Switching to fill mode fills the canvas with selected color
    • Select among 9 colors to fill or paint with
    • Download the artwork you created
